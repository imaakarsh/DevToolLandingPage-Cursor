
# Cursor Clone Landing Page

A static landing page clone inspired by Cursor's marketing site. This project focuses on layout, typography, and section composition using plain HTML and CSS.

## Features

- Sticky navigation with primary and secondary actions
- Hero section with download CTA and product screenshot
- Trusted-by brand strip
- Alternating feature blocks with imagery
- Testimonials grid
- Frontier highlights and changelog cards
- Recent highlights section and final CTA
- Multi-column footer

## Recreated Sections

- Sticky navigation bar with feature links and CTA buttons
- Hero headline with primary download button
- Trusted-by logos strip
- Three alternating feature blocks with imagery
- Testimonials grid with author cards
- Frontier highlights (three cards)
- Changelog cards and footer link
- Recent highlights posts list
- Final download promotion banner
- Multi-column footer

## Fonts and Colors Used

### Fonts

- CursorGothic (custom font loaded via `@font-face`)
- Weights: 300 (light) and 400 (regular)

### Colors

- Background: #13120b
- Secondary background: #1b1913
- Primary text: #ffffff
- Accent: #f54e00
- Border: 0.5px solid gray

## Tech Stack

- HTML5
- CSS3 (custom properties, flexbox, responsive layout)

## Getting Started

1. Open the folder in VS Code.
2. Open `index.html` in a browser (or use Live Server).

## Notes

- Fonts are loaded via `@font-face` in CSS.
- Images are referenced from the `Assets/images` folder.
